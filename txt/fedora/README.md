# Fedora Installation Script

This folder contains a comprehensive Fedora installation script tailored for systems with AMD Ryzen 5700X and Nvidia 5060 hardware. The script automates the setup of essential tools, drivers, and applications for general use, gaming, and content creation.

## Features

- **System Updates**: Ensures the system is up-to-date with the latest packages.
- **Essential Tools**: Installs a variety of utilities for productivity and system management.
- **Driver Setup**: Configures AMD and Nvidia drivers for optimal performance.
- **Multimedia Support**: Adds codecs, libraries, and repositories for enhanced multimedia capabilities.
- **Flatpak Integration**: Sets up Flatpak and installs popular applications.
- **Firmware Installation**: Installs various firmware packages for hardware compatibility.
- **Performance Optimization**: Tunes the system for AMD Ryzen processors.
- **Error Handling**: Includes robust error handling and logging for troubleshooting.

## Usage

1. Clone this repository or download the script.
2. Ensure the script has executable permissions:
   ```bash
   chmod +x fedora-install-script.sh
   ```
3. Run the script with sudo privileges:
   ```bash
   sudo ./fedora-install-script.sh
   ```
